# python_c_api
Example of use of Python-C API

## Instructions

- Create *Python* module from the *C* one with `python3 setup__my_module.py
    install` (may need `sudo` rights on *Linux*)
- Include it in your *Python* project with `include my_module` or `from
    my_module import *`
